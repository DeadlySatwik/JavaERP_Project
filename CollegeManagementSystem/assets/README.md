Assets folder

Place the following images in this folder with the exact filenames used by the app:

- background.jpg  (College Background Theme)
  Recommended: https://hips.hearstapps.com/hmg-prod/images/berry-college-historic-campus-at-twilight-royalty-free-image-1652127954.jpg

- student_placeholder.jpg (Student Picture placeholder)
  Recommended: https://t3.ftcdn.net/jpg/08/94/18/56/360_F_894185679_2ZvyxKySzDZ8axuP2mC96NcL2Kbb188I.jpg

- teacher_placeholder.jpg (Teacher Picture placeholder)
  Recommended: https://thumbs.dreamstime.com/b/professor-23802534.jpg

If you don't add images, the app will attempt to load them from the remote URLs at runtime.
